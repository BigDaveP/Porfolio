Projets

Mon Carnet de Santé

Projet réalisé avec le CAAPBSL pour l'intégration web du Carnet de Santé dans le cadre d'un projet avec l'École du Web. Le projet a été finalisé durant l'été par mes soins.

Le projet inclue :

  • Une ergonomie adaptée au personne ayant des problèmes de littéracie.
  • Un système de gestion de compte.
  • Un système de gestion de rendez-vous.
  • Plusieurs plugins réalisés "from scratch" pour avoir des éléments adaptés aux utilisateurs.

Réalisé avec :

  • ReactJS + ViteJS
  • Laravel (API)
  • MailTrap
